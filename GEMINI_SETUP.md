# Gemini API Setup for ATS Score Checker

The ATS Score Checker now uses Google's Gemini AI to provide accurate, intelligent resume analysis against job descriptions.

## Setup Instructions

### Step 1: Get Your Gemini API Key

1. Visit [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Sign in with your Google account
3. Click "Create API Key"
4. Select or create a new project
5. Copy your API key

### Step 2: Add API Key to Code Nectous

1. Go to your Code Nectous account page: **Account Settings**
2. Scroll to **"ATS Checker Settings"**
3. Paste your Gemini API key in the input field
4. Click **"Save API Key"**

The API key is stored securely in your browser's local storage (not sent to our servers).

### Step 3: Use the ATS Checker

1. Go to the home page
2. Scroll down to **"ATS Score Checker"**
3. Upload your resume (PDF, DOCX, or TXT)
4. Paste the job description
5. Click **"Analyze Resume"**
6. Wait for Gemini to analyze your resume
7. Get your ATS score and personalized suggestions

## What You Get

- **ATS Compatibility Score**: 0-100% rating of how well your resume matches the job
- **Keyword Analysis**: Which job skills you have and which you're missing
- **Formatting Score**: How well your resume is structured for ATS systems
- **Smart Suggestions**: AI-generated recommendations to improve your resume

## API Key Security

- Your API key is stored only in your browser's local storage
- It's never sent to our servers (all API calls go directly to Google)
- Your resume is not stored - analysis happens in real-time
- You can remove the API key anytime in Account Settings

## Troubleshooting

**Error: "API Key Missing"**
- Make sure you've saved your Gemini API key in Account Settings

**Error: "Analysis failed"**
- Check that your API key is valid
- Ensure your Google Cloud project has Generative AI API enabled
- Make sure your resume and job description are not empty

**How to Show/Hide the API Key**
- Check the "Show API Key" checkbox to see your saved key
- Uncheck to hide it again for security
